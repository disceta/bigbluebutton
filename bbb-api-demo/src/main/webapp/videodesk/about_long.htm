Общие положения

 

Сервис Videodesk-   это простой и удобный инструмент для проведения WEB-презентаций во время телефонных переговоров или аудио конференций.

При этом сам разговор  между участниками WEB-презентаций осуществляется вне сервиса Videodesk, с помощью любых видов связи.

Сервис может использоваться с целью  увеличения продаж товаров и услуг, как дополнение к контакт-центрам, для дистанционного обучения  и  т.д.

Клиентам Videodesk  предоставляется персональный  WEB-кабинет, в котором они могут проводить презентации практически без ограничения количества участников.

 

Участники WEB –  презентации разделяются на три типа.

  • Лидер
  • Докладчик
  • Участник

Лидер – производит управление презентацией, загружает, удаляет файлы для презентации, производит обмен текстовыми сообщениями с другими участниками презентации, назначает докладчика. Лидер может выступать в роле докладчика.

 

 Докладчик – участник презентации, которому модератор во время презентации предоставил дополнительные возможности. Докладчик может загружать и удалять файлы для презентации, вести презентацию.

 

Участник – может производить обмен текстовыми сообщениями с другими участниками презентации. 

 

В одной WEB -презентации может быть несколько лидеров.

 

Вход в презентацию лидера производится по дополнительному ПИН коду.

 

WEB - презентация всегда проводится с лидером.

Передача статуса «ведущий» другому участнику

 

Устанавливать статус участника может только модератор презентации.

Для этого модератор выбирает в списке участников того участника, которому  необходимо установить статус ведущего и нажмает клавишу «Переключить в …».

 

 

К выбранному участнику перейдёт статус ведущего и у него появится дополнительные инструменты для управления файлами.

Участник, который был «ведущим» станет просто участником презентации.

 

 

Обмен текстовыми сообщениями (чат).

Сообщения для всех участников

Каждый из участников конференции может послать текстовое сообщение одновременно всем участникам презентации.

Для этого при открытой закладке без названия необходимо набрать текст в окне сообщений, и нажать кнопку «Отправить».

 

У всех участников презентации на экране отобразится переданное текстовое сообщение.

 

 

Сообщения для отдельного участника

Каждый из участников конференции может послать текстовое сообщение любому конкретному участнику презентации.

Для выбора участника для обмена текстовыми сообщениями выберите вкладку с обозначением «+».

 

На вкладке помещён список участников конференции. Выберите участника, для которого необходимо послать текстовое сообщение.

 

На форме появится новая вкладка с именем выбранного участника презентации.

Откройте эту вкладку, напишите текстовое сообщение и нажмите кнопку «Отправить».

 

 

На закладке с имением участника будет отображаться личная переписка.

У участника презентации, кому было послано личное сообщение, появится новая вкладка с именем участника пославшего это сообщение.

Для прочтения личного сообщения откройте закладку с именем участника.

На данной закладке будет отображена вся переписка с этим конкретным участником.
